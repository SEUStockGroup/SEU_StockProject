# SEU_StockProject
东南大学沪深股市数据分析

# 初始化：项目目录结构

```
├─.idea
│  └─inspectionProfiles
├─src //source源文件
│  ├─assets //项目资源文件
│  ├─components //组件文件
│  ├─router //路由文件
│  ├─utils //工具文件
│  └─views //视图文件
├─static //静态文件 
├─templates //模板文件
└─venv //虚拟环境（未上传到github上）
```

# 一些Git命令

```bash
//克隆项目
git clone
git remote -v
//创建分支
git branch ""
git checkout
//添加代码
git add *
git commit -m ""
git push

//拉取分支
git fetch 远程代理名 服务器分支名 (git fetch origin teacher)
//创建切换并映射分支
git checkout -b 本地分支名  远程代理名/服务器分支名    ：建议本地分支名=服务器分支名（git checkout -b teacher origin/teacher）
```
